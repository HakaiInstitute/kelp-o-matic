# Beginner Guide

Welcome! This guide is designed to take you from having **zero command line experience** to successfully processing
geospatial imagery with Habitat-Mapper.

We recommend following these modules in order.

<div class="grid cards" markdown="1">

-   :material-console: **1. Terminal Crash Course**

    ---
    Start here if you are new to the command line. Learn how to navigate folders and run basic commands.

    [:material-arrow-right: Start Learning](terminal_crash_course.md)

-   :material-download: **2. Installation**

    ---
    Set up `uv` (our environment manager) and install the Habitat-Mapper software on your computer.

    [:material-arrow-right: Install Now](../installation.md)

-   :material-play: **3. Processing Images**

    ---
    Learn how to use the `hab` command to detect kelp, mussels, or other species in your imagery.

    [:material-arrow-right: Run Tool](execution.md)

-   :material-map-check: **4. Post-Processing**

    ---
    Take your raw model outputs and turn them into clean, analysis-ready shapefiles using ArcGIS Pro or QGIS.

    [:material-arrow-right: Clean Data](post_processing.md)

</div>

!!! tip "Need Help?"
    If you get stuck at any point, please check the [FAQs](../faqs.md) or open an issue on our [GitHub Repository](https://github.com/HakaiInstitute/habitat-mapper/issues).
