Features
--------

- Kelp detection in RGB RPAS imagery
- Mussel detection in RGB RPAS imagery

Installation
------------
The most reliable way to install hakai_segmentation is with Anaconda.

.. code-block:: bash

    conda install -c hakai-institute hakai-segmentation
